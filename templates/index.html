Welcome to mvt server

A simple, fast, and powerful server for vector tiles distribution.

With mvt server, you can:

  • Serve high-performance vector layers.
  • Manage map styles with an integrated style server.
  • Handle users and groups with flexible access control.
  • Coming soon: A metadata server compliant with ISO 19115 for spatial data documentation.

Explore the available layers in the "Catalog" menu and customize their appearance in the "Styles" section.
